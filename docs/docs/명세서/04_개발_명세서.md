## 1. Quick Start (5분 셋업 가이드)

### 1.1 사전 요구사항

```bash
# 필수
rustc --version    # 1.70+
python3 --version  # 3.x
node --version     # 18+
docker --version   # Docker Desktop

# macOS 전용: Accessibility 권한 허용
# 시스템 설정 → 개인정보 보호 및 보안 → 손쉬운 사용 → 터미널 앱 허용
```

### 1.2 3단계 셋업

```bash
# Step 1: 환경변수 설정
cd mac/core
cp .env.example .env
# .env 파일을 열어 OPENAI_API_KEY 입력

# Step 2: DB 초기화 + Core 빌드
cd mac
PYTHONPATH=src python scripts/init_db.py --config configs/config.yaml
cd core && cargo build --release

# Step 3: 실행
# 터미널 1 — Core API
cargo run --release

# 터미널 2 — Collector
cd mac && PYTHONPATH=src python -m collector.main --config configs/config.yaml

# 터미널 3 — n8n (선택)
docker compose up -d

# 터미널 4 — Frontend (개발 모드)
cd mac/web && npm install && npm run dev
```

### 1.3 실행 확인

```bash
# Core API 헬스체크
curl http://127.0.0.1:5680/health

# Collector 헬스체크
curl http://127.0.0.1:8080/health

# n8n 상태 확인
curl http://127.0.0.1:5678/healthz

# 간단한 NL 테스트
curl -X POST http://127.0.0.1:5680/api/agent/intent \
  -H "Content-Type: application/json" \
  -d '{"text": "오늘 날씨 알려줘"}'
```

## 11. 트러블슈팅 가이드

### 11.1 빌드 에러

| 증상 | 원인 | 해결 |
|------|------|------|
| `error[E0433]: failed to resolve` | Rust 모듈 미등록 | `lib.rs`에 `pub mod <module>;` 추가 |
| `linker 'cc' not found` | C 컴파일러 미설치 | macOS: `xcode-select --install`, Linux: `apt install build-essential` |
| `rusqlite bundled build failed` | SQLite 빌드 실패 | `cargo clean` 후 재빌드 |
| `npm run build` 실패 | TypeScript 타입 에러 | `npm run lint` 로 에러 위치 확인 |
| Tauri 빌드 실패 | WebView 미설치 | macOS: 자동, Windows: WebView2 Runtime 설치 |

### 11.2 런타임 에러

| 증상 | 원인 | 해결 |
|------|------|------|
| `OPENAI_API_KEY not set` | 환경변수 누락 | `core/.env`에 Key 설정 |
| `Address already in use :5680` | 포트 충돌 | 기존 프로세스 종료: `kill.sh` 또는 `STEER_API_PORT` 변경 |
| `database is locked` | SQLite 동시 접근 | busy_timeout 확인, 중복 프로세스 확인 |
| `Accessibility permission denied` | OS 권한 부족 | 시스템 설정 → 손쉬운 사용 → 앱 허용 |
| `n8n connection refused :5678` | Docker 미실행 | `docker compose up -d` |
| `singleton lock: another instance` | 중복 실행 | 기존 인스턴스 종료 또는 `STEER_LOCK_DISABLED=1` |

### 11.3 데이터 문제

| 증상 | 원인 | 해결 |
|------|------|------|
| 이벤트 수집 안됨 | 센서 미실행 또는 Denylist 앱 | `privacy_rules.yaml` allowlist 확인 |
| DB 크기 급증 | 리텐션 미실행 | `python scripts/run_retention.py` 수동 실행 |
| 패턴 미감지 | 데이터 부족 (<3일) | 충분한 데이터 축적 대기 또는 `seed_dummy_data.py` |
| DB 경로 불일치 | Core와 Collector DB 경로 상이 | `/api/system/db-paths` 확인 후 경로 통일 |
